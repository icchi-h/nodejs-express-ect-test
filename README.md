# Expressを使い自動生成したwebサイトサンプル
node.jsを利用したWebアプリ開発のテスト用

* node.js
* express
* ect

## 実行
```bash
$ git clone <repository url>
$ cd path/to/repository
$ npm install # 利用するnode.jsのパッケージをインストール (package.jsonに記録)
$ npm run start
```
